# goit-algo-fp

Фінальний проєкт з курсу алгоритмів GoIT.

## Структура проєкту

| Файл | Завдання |
| --- | --- |
| `task1.py` | Однозв'язний список: реверс, сортування та злиття відсортованих списків |
| `task2.py` | Рекурсивна побудова фрактала "дерево Піфагора" |
| `task3.py` | Алгоритм Дейкстри з використанням бінарної купи |
| `task4_and_task5.py` | Візуалізація бінарної купи та обходів дерева |
| `task6.py` | Жадібний алгоритм і динамічне програмування для вибору їжі |
| `task7.py` | Метод Монте-Карло для кидання двох кубиків |

## Відповідність вимогам

### Завдання 1. Однозв'язний список

- Реалізовано реверс однозв'язного списку зі зміною посилань між вузлами.
- Реалізовано сортування однозв'язного списку методом бульбашки.
- Реалізовано злиття двох відсортованих однозв'язних списків в один відсортований список.

### Завдання 2. Рекурсія

- Програма рекурсивно будує фрактал "дерево Піфагора".
- Користувач може вказати рівень рекурсії.

### Завдання 3. Алгоритм Дейкстри

- Створено зважений граф.
- Для вибору наступної вершини використовується `heapq`, тобто бінарна купа.
- Обчислюються найкоротші відстані від початкової вершини до всіх інших вершин графа.

### Завдання 4. Візуалізація бінарної купи

- Масив перетворюється на мінімальну купу за допомогою `heapq.heapify`.
- Із купи створюється бінарне дерево.
- Дерево візуалізується за допомогою `networkx` та `matplotlib`.

### Завдання 5. Візуалізація обходів дерева

- Реалізовано обхід у ширину BFS із використанням черги.
- Реалізовано обхід у глибину DFS із використанням стеку.
- Кожен вузол під час обходу отримує колір у форматі HEX RGB.
- Кольори змінюються від темних до світлих відтінків відповідно до порядку обходу.

### Завдання 6. Жадібний алгоритм і динамічне програмування

- Реалізовано функцію `greedy_algorithm`.
- Реалізовано функцію `dynamic_programming`.
- Обидві функції вибирають страви в межах заданого бюджету для максимізації калорійності.

### Завдання 7. Метод Монте-Карло

- Імітується велика кількість кидків двох кубиків.
- Обчислюються ймовірності сум від `2` до `12`.
- Результати Монте-Карло порівнюються з аналітичними ймовірностями.
- Порівняння виводиться у таблиці та відображається на графіку.
- У кінці програма виводить короткий висновок із максимальним відхиленням.

## Висновок для методу Монте-Карло

Метод Монте-Карло дає результати, близькі до аналітичного розподілу
ймовірностей для двох кубиків. За малої кількості кидків випадкові відхилення
помітніші. Коли кількість кидків зростає, експериментальні ймовірності
наближаються до теоретичних значень: сума `7` є найімовірнішою, а суми `2` і
`12` - найменш імовірними.

## Як запустити

Встановіть залежності:

```bash
pip install -r requirements.txt
```

Запустіть потрібне завдання:

```bash
python task1.py
python task2.py
python task3.py
python task4_and_task5.py
python task6.py
python task7.py
```
